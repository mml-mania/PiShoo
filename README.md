PiShoo
======================
PiStarter用。  
昔あった「連射測定機能つき時計」の、連射測定機能部分です。  
手持ちのジョイパッドの連射性能の測定などにもいかがでしょうか。

遊び方
------
### SHOOTING A ###
Aボタンが押された時から10秒間、Aボタンの連射回数を測定します。  
痙攣撃ちモード。

### SHOOTING A/B ###
A/Bボタンが押された時から10秒間、A/Bの連射回数を測定します。  
ピアノ撃ちモード。  

説明など
----------------
- 「ウォッチ」側の機能はありません。
- Xinput方式のジョイパッドを接続してください。  
DirectInput方式のジョイパッドでも、A/Bボタンが認識できれば測定は可能なはずです。メニュー選択はキーボードが必要だと思います。
- SHOOTING Aは60連射/秒、SHOOTING A/Bは30連射/秒まで測定可能です。
- いやが上にも連射しなければならない気分を盛り上げるBGMなどはありません。
- 100連射を超えたりするとアンロックされるミニゲームはありません。
- DSにあったようなグラフ描画機能はありません。
